# GitLab CLI Snippets

## Clone repository

```
glab repo clone <@owner>/<@repository>
```

## Create merge request

```
glab mr create
```

## View merge request in browser

```
glab mr view --web <@mr_number>
```

## Check out merge request locally

```
glab mr checkout <@mr_number>
```
