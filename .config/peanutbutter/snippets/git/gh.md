# GitHub CLI Snippets

## Clone repository

```bash
gh repo clone <@owner>/<@repository>
```

## View pull request in browser

```bash
gh pr view --web <@pr_number>
```

## Check out pull request locally

```bash
gh pr checkout <@pr_number>
```

## Check pull request status

```bash
gh pr status
```
